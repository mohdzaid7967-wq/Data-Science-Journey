# Python Dictionary

This folder contains beginner-friendly Python programs based on **Dictionary**.

## 📖 About

A dictionary is a built-in Python data structure used to store data in **key-value pairs**. Dictionaries are mutable, unordered, and allow fast data retrieval using keys.

### Syntax

```python
dictionary = {
    "key": "value"
}
```

---

## 📂 Programs Included

### 1. count_frequency.py
- Counts the frequency of elements using a dictionary.

### 2. employe_salary.py
- Stores employee names and salaries using a dictionary.

### 3. find_key.py
- Checks whether a key exists in a dictionary.

### 4. marks.py
- Stores students' marks and displays their details.

### 5. merge_dic.py
- Merges two dictionaries into one.
