# If, Elif and Else in Python

This folder contains decision-making programs using if, elif and else.

## Programs

- BMI.py - BMI Calculator
- electricity_bill.py - Electricity Bill Calculator
- leap_year.py - Leap Year Checker
- odd_even_weird.py - Odd Even Weird (HackerRank)

## Concepts Covered

- if
- elif
- else
- Comparison operators
- Logical operators
