# Python Tuple

This folder contains beginner-friendly Python programs based on **Tuples**.

## 📖 About

A tuple is an ordered and immutable collection of elements. Tuples are used to store fixed data securely.

## 📂 Programs Included

### 1. Tuple to List Conversion
- Converts a tuple into a list, adds an element, and converts it back into a tuple.

### 2. Count Element Occurrences
- Counts how many times an element appears in a tuple.

### 3. Element Existence Check
- Checks whether a given element exists in a tuple.

### 4. Maximum Element Without `max()`
- Finds the largest element in a tuple without using the `max()` function.
