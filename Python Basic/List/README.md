# Python List

This folder contains beginner-friendly Python programs based on **Lists**.

## 📖 About

A list is an ordered and mutable collection in Python. Lists are one of the most commonly used data structures for storing and manipulating data.

## 📂 Programs Included

### 1. Count Even and Odd Numbers
- Counts the total number of even and odd numbers in a list.

### 2. Largest Number Without `max()`
- Finds the largest number in a list without using the `max()` function.

### 3. Remove Duplicates
- Removes duplicate elements while preserving the original order.

### 4. Reverse a List
- Reverses a list without using `reverse()` or slicing.

### 5. Second Largest Number
- Finds the second largest element in a list without using built-in functions.
