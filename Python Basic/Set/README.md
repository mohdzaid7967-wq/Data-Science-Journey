# Python Set

This folder contains beginner-friendly Python programs based on **Sets**.

## 📖 About

A set is an unordered collection of unique elements. It is useful for removing duplicates and performing mathematical set operations.

## 📂 Programs Included

### 1. Find Common Elements
- Finds the common elements between two sets using intersection.

### 2. Unique Words in a Sentence
- Takes a sentence as input and prints only unique words using a set.

### 3. Difference Between Two Sets
- Finds the elements present in the first set but not in the second.

### 4. Subset Check
- Checks whether one set is a subset of another.

### 5. Union Without `|` Operator
- Finds the union of two sets without using the `|` operator.
