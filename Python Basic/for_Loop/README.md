# For Loop in Python

This folder contains beginner Python programs based on the **for loop**.

## Programs

- table.py - Print the multiplication table of a number.
- sum_of_even.py - Find the sum of even numbers.
- vowels.py - Count vowels in a string.

## Concepts Covered

- for loop
- range()
- if statement
- Arithmetic operations
