# Python Functions

This folder contains beginner-friendly Python programs based on **Functions**.

## 📖 About

A function is a reusable block of code that performs a specific task. Functions make programs modular, readable, and easier to maintain.

### Syntax

```python
def function_name(parameters):
    # Code
    return value
```

---

## 📂 Programs Included

### 1. even_odd.py
- Checks whether a number is even or odd using a function.

### 2. factorial.py
- Calculates the factorial of a number using a function.

### 3. largest_num.py
- Finds the largest number between two numbers using a function.
