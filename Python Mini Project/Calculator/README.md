# Calculator

A simple command-line calculator built using Python.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Error handling using try-except

## Technologies Used

- Python 3
- Match-Case Statement
- Exception Handling

## How to Run

1. Open terminal
2. Run the program:

```bash
python calculator.py
```

## Example

Input:

```text
Enter the first number: 10
Enter the second number: 5
Enter the Operation: +
```

Output:

```text
The result is: 15
```
