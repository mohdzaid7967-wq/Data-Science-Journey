# Inheritance in Python

This folder demonstrates **Inheritance**, which allows one class to inherit the properties and methods of another class.

## Topics Covered

- Single Inheritance
- Parent Class
- Child Class
- super() Method

## Files

- `animal.py`
- `car.py`
- `super_method.py`

## Learning Outcome

- Reuse code using inheritance.
- Understand parent and child classes.
- Learn how to use the `super()` method.
