# Encapsulation in Python

This folder contains examples of **Encapsulation**, one of the four pillars of Object-Oriented Programming (OOP).

## Topics Covered

- Public Members
- Protected Members
- Private Members
- Data Hiding

## Files

- `public_modifier.py` – Example of public variables and methods.
- `protected_modifier.py` – Example of protected members using `_`.
- `private_protecting_method.py` – Example of private members using `__`.

## Learning Outcome

- Understand data hiding.
- Learn access modifiers in Python.
- Protect object data using encapsulation.
