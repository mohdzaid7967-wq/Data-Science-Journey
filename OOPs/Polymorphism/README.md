# Polymorphism in Python

This folder contains examples of **Polymorphism**, one of the four pillars of Object-Oriented Programming (OOP).

## What is Polymorphism?

Polymorphism means **"many forms."**

It allows the same method or operator to behave differently depending on the object or data it is working with.

For example:

- A `Dog` object and a `Cat` object can both have a `sound()` method, but each produces different output.
- The `+` operator can add numbers as well as concatenate strings. This is known as **Operator Overloading**.
